## 2026-05-01

- [期末处理] 重写「期末处理」章节，扩展为 6 个子页签：参数管理、启用管理、库存关账、项目关账、期末结账、顺延查询；新增全局交互规范（Toast、二次确认、行高亮、执行结果日志区）与公用 FAQ 弹窗（4 条问答）
- [期末处理] 调整子页签顺序为「参数管理 → 启用管理 → 库存关账 → 项目关账 → 期末结账 → 顺延查询」，`参数管理` 移到 `启用管理` 之前
- [期末处理/参数管理] 新增「参数管理」子页签，分两组：①项目核算关账控制（核算前 EBS 数据获取校验、收款计税校验等开关与校验时间）；②核算定时任务时间设定（自动核算/同步触发时间）
- [期末处理/启用管理] 新增「启用管理」子页签，按财务主体启用核算、库存关账、项目关账、期末结账等模块；定义启用前置条件与启用后效果
- [期末处理/库存关账] 新增「库存关账」子页签：列表 + 选中关账 + 二次确认 + 关账批次号 + 顺延机制（未完成单据展示与跳转）
- [期末处理/项目关账] 新增「项目关账」子页签：列表 + 选中关账 + 二次确认 + 关账校验（库存须先关账等）+ 顺延机制
- [期末处理/期末结账] 新增「期末结账」子页签：财务主体 + 期间结账，校验前置（库存关账、项目关账完成）+ 顺延机制
- [期末处理/顺延查询] 新增「顺延查询」子页签：按关账批次号查询顺延履历，明细弹窗展示被顺延的单据及顺延原因
- [集成管理/同步日志] 列表 `请求参数` 列改为以 `查看` 按钮形式呈现，点击打开 `请求参数（弹窗）`
- [集成管理/同步日志] 新增 `请求参数（弹窗）`：展示 `来源单号`、`接口类型`、`参数内容（JSON）`，提供 `复制`、`关闭` 按钮，支持点击遮罩、`Esc` 键关闭

## 2026-04-30

- [集成管理] 新增 `集成管理` 模块章节，包含 `EBS映射配置`（含 `财务主体映射`、`内部往来映射` 子页面）与 `同步日志` 子章节
- [集成管理/EBS映射配置] 维护 `保存`、`取消` 按钮，列表字段含 `中台财务主体`、`EBS公司编码`、`EBS公司id`（锁定带出）、`操作人`、`操作时间`，行内 `新增行`、`删除行` 操作

- [集成管理/同步日志] 列表字段含 `来源单号`、`主合同号`、`接口类型`、`状态`、`同步时间`、`最后同步时间`、`请求参数`、`反馈信息`，行内提供 `重试` 操作
